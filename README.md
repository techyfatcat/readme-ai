## 📖 Overview

README-AI is an open-source CLI tool that analyzes your project and automatically generates a clean, well-structured `README.md`.

Instead of manually writing documentation, simply run the CLI and let README-AI inspect your project structure, technologies, build tools, runtime, deployment configuration, environment variables, and more.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/readme-ai.git
```

Move into the project

```bash
cd readme-ai
```

Install dependencies

```bash
npm install
```

Run in development

```bash
npm run dev
```

Build the project

```bash
npm run build
```
